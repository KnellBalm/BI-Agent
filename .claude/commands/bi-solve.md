# /bi-solve — BI 분석 오케스트레이터

비즈니스 문제를 말하면 분석 방향을 잡아드립니다.

## 실행

이 스킬을 실행하기 전에 `skills/bi-solve.md` 파일을 읽고 그 안의 지침을 정확히 따른다.

## 시작 방법

`$ARGUMENTS`가 있으면 그것을 사용자의 문제로 받아 G1부터 시작한다.
`$ARGUMENTS`가 없으면: "어떤 비즈니스 문제를 분석하고 싶으신가요?"

## 주의사항

- Gate를 절대 건너뛰지 않는다
- 한 번에 하나의 질문만 한다
- 사전 질문지 파일 생성 없이 분석을 시작하지 않는다
