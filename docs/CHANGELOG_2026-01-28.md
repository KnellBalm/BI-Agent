# CHANGELOG - 2026-01-28

오늘(2026-01-28) 진행된 BI-Agent의 주요 변경 사항 및 기술적 조치 내역입니다. 다음 릴리즈(v1.0.1) 배포 시 참고해 주세요.

---

## 🎨 UI/UX 개선 (Entrance Hall & Command Palette)

*   **로고 개편**: 복잡한 ASCII 아트 대신 가독성이 뛰어나고 전문적인 **'BI - Agent'** 텍스트 배너 디자인을 적용했습니다.
*   **슬래시 명령어(/) 복구**:
    *   `/` 입력 시 명령어 팔레트가 입력창 상단에 정확한 위치에 팝업되도록 CSS 및 로직을 수정했습니다.
    *   방향키(`Down`)를 통해 입력창에서 팔레트로 포커스를 이동하고 명령어를 선택할 수 기능을 추가했습니다.
*   **부팅 로그 가시화**: TUI 실행 시 가려져서 보이지 않던 초기화 로그(모델 설정, 프로젝트 로드 등)를 채팅창 상단에 직접 출력하여 사용자에게 현재 상태를 명확히 전달합니다.

## 🔐 인증 및 보안 (Auth Management)

*   **지능형 인증 감지**: 실행 시 자동인증 여부를 체크하여, 인증이 필요한 경우에만 `AuthScreen`을 즉시 띄우도록 로직을 고도화했습니다.
*   **환경 변수 연동**: `credentials.json` 파일이 없더라도 시스템 환경 변수(`GEMINI_API_KEY`, `ANTHROPIC_API_KEY`)가 설정되어 있다면 자동으로 인증된 것으로 간주하는 Fallback 로직을 추가했습니다.
*   **보안 강화**: `.gitignore` 파일에 개인용 연결 설정(`connections.json`) 및 로컬 데이터 저장 폴더를 추가하여 보안 사고를 예방했습니다.

## 📦 설치 및 배포 자동화 (Distribution & CI/CD)

*   **PyPI 설치 트러블슈팅**:
    *   Python 3.10+ 버전 요구사항에 따른 설치 실패 원인을 분석하고 해결 지침을 마련했습니다.
    *   동료 PC 등의 '파이썬 버전 꼬임' 현상에 대비한 `python3 -m pip install` 공식 가이드를 완성했습니다.
*   **GitHub Actions 수동 배포 구축**:
    *   `.github/workflows/pypi-publish.yml`을 통해 프로젝트 배포 자동화를 구현했습니다.
    *   **이중 안전 장치**: 사용자가 직접 "yes"를 입력하거나 태그(`v*`)를 푸시했을 때만 실제 배포가 이루어지도록 설계했습니다.
    *   사용자가 지정한 GitHub Secret(`PYPI_API`)을 사용하도록 연동 완료했습니다.
