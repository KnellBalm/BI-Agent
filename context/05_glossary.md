# 비즈니스 용어사전

> 이 사전의 용어를 기준으로 모든 분석 결과와 리포트가 작성됩니다.

## 📌 비즈니스 핵심 용어

| 용어 | 정의 | 관련 지표 | 비고 |
|------|------|----------|------|
| 활성 사용자 | 최근 30일 내 1회 이상 서비스를 이용한 사용자 | MAU, DAU | 로그인 기준 |
| 전환 | 무료 회원이 유료 플랜으로 업그레이드하는 행위 | 전환율, CAC | |
| 이탈 (Churn) | 유료 구독을 취소하거나 30일 이상 미접속한 상태 | Churn Rate | |
| 코호트 | 동일 기간에 가입한 사용자 그룹 | 리텐션율 | |
| LTV | 고객이 서비스를 이용하는 전체 기간 동안 발생시키는 예상 총 수익 | ARPU, Churn | |
| 퍼널 | 사용자가 특정 목표를 달성하기까지의 단계별 흐름 | 전환율 | |
| 리텐션 | 특정 시점에 가입한 사용자가 이후에도 서비스를 계속 이용하는 비율 | MAU, DAU | |

## 🔢 약어 사전

| 약어 | 풀네임 | 설명 |
|------|--------|------|
| MAU | Monthly Active Users | 월간 활성 사용자 수 |
| DAU | Daily Active Users | 일간 활성 사용자 수 |
| MRR | Monthly Recurring Revenue | 월간 반복 수익 |
| ARR | Annual Recurring Revenue | 연간 반복 수익 |
| ARPU | Average Revenue Per User | 사용자당 평균 수익 |
| AOV | Average Order Value | 평균 주문 금액 |
| CAC | Customer Acquisition Cost | 고객 획득 비용 |
| LTV | Lifetime Value | 고객 생애 가치 |
| ROAS | Return On Ad Spend | 광고 수익률 |
| GMV | Gross Merchandise Value | 총 거래액 |
| CTR | Click-Through Rate | 클릭률 |
| CVR | Conversion Rate | 전환율 |
| NPS | Net Promoter Score | 순 추천 지수 |
