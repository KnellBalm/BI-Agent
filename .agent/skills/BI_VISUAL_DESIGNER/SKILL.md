# BI Visual Designer 스킬 가이드

이 스킬은 데이터 조여 결과를 어떻게 시각적으로 표현하고, 기존 리포트의 레이아웃과 조화를 이룰지 결정하는 지침을 제공합니다.

## 1. 핵심 책임
- **Visual Decoding**: `VisualDecoder`를 사용하여 `etc` 필드 내의 Base64 데이터를 완벽히 해석하고 수정합니다.
- **차트 선택 자동화**: 데이터의 특성(Categorical vs Numeric)에 따라 가장 적합한 위젯 타입(Bar, Line, Pie, Table)을 추천합니다.
- **스타일 가이드 준수**: 리포트 전체의 색상 테마, 폰트 크기, 여백 등을 일관되게 유지합니다.

## 2. 세부 가이드라인

### [A] 데이터-위젯 바인딩
- 조회된 데이터의 컬럼명이 위젯의 `pivot`이나 `metric` 필드와 어떻게 매핑되는지 정의합니다.
- `elementId`를 기반으로 정교하게 타겟 위젯을 식별합니다.

### [B] 지능형 레이아웃 배치
- 새로운 위젯을 추가할 때 기존 위젯들과 겹치지 않도록 `position` 값을 계산합니다.
- 빈 공간을 찾아 최적의 `size`로 자동 배치합니다.

### [C] 메타데이터 보정
- `title`, `description` 등 텍스트 요소를 데이터의 특징에 맞게 자동 생성하여 리포트의 완성도를 높입니다.
