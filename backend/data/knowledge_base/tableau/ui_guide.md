# Tableau Filter UI Guide
To apply a filter to a worksheet in Tableau:
1. Drag the field you want to filter by to the **Filters** shelf.
2. In the Filter dialog box, select the values you want to include or exclude.
3. Click **OK**.
4. To show the filter to users, right-click the field on the Filters shelf and select **Show Filter**.

# Tableau Parameter Guide
To create a parameter:
1. In the Data pane, click the drop-down arrow in the upper right corner and select **Create Parameter**.
2. Name the parameter and specify the data type.
3. Define the allowable values (All, List, or Range).
4. Use the parameter in a calculated field or filter.
