# Power BI DAX Guide
To create a new measure in Power BI:
1. Go to the **Modeling** tab in the ribbon.
2. Click **New Measure**.
3. In the formula bar, enter the DAX expression (e.g., `Total Sales = SUM(Sales[Amount])`).
4. Press Enter to confirm.

# Power BI Visualization Guide
To add a visual to a report:
1. Open the **Visualizations** pane on the right.
2. Click on the icon of the visual you want to add (e.g., Bar chart, Line chart).
3. Drag fields from the **Fields** pane into the visual's data buckets (Axis, Legend, Values).
